# AI Research Intelligence Agent

A Multi-Agent AI Research Paper Analysis System built using n8n and Google Gemini AI.

## Features

* Research Paper Text Extraction
* Executive Summary Generation
* Methodology Analysis
* Dataset Extraction
* Interview Question Generation
* Future Research Recommendations
* Automated Research Insights

## Workflow Architecture

Research Paper PDF
↓
Extract Text
↓
Research Summarizer Agent
Methodology Analysis Agent
Dataset Extraction Agent
Interview Question Agent
Future Research Agent
↓
Research Insights

## Tech Stack

* n8n
* Google Gemini AI
* JavaScript
* PDF Text Extraction
* Workflow Automation

## Project Structure

* workflow.json → n8n workflow export
* screenshots/ → Project screenshots
* README.md → Project documentation
